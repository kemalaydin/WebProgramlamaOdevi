Hakkımda

Kemal AYDIN

1995 Yılında İzmir'de doğdum. Liseyi Mithatpaşa Anadolu Teknik Lisesi, Bilişim Teknolojileri Bölümünde okudum. Daha sonra Ege Üniversitesi, Bilgisayar Programcılığı bölümünü kazandım. Burayı 3.21 ortalama ile bölüm birincisi olarak bitirdim. Ardından Dikey Geçiş Sınavı ile Erciyes Üniversitesi, Bilgisayar Mühendisliği bölümünü kazandım. Buradaki eğitimim 4. sınıf olarak devam etmektedir. İlk programımı ilkokul 4. sınıf öğrencisi iken İzmir Adliyesi'ne yaptım. Bu program ile otomatik bilirkişi raporları oluşturulabiliyordu. Lise yıllarında ise Stajyeri.com adında bir web projesi geliştirdim. Bu proje ile EGİAD, E-Tohum, Ankara Bilkent Startup Zirvesi, İstanbul Üniversitesi, Startup Turkey gibi etkinliklerde dereceler aldım. Aktif olarak bir şirkette Full-Stack developer olarak çalışmaktayım. Firma bünyesinde PHP, JavaScript ( Node.JS, ExpressJS, React ) dilleri ile ilgileniyorum. Ayrıca Python, CSS ve HTML dillerine ileri seviyede, Python, Ruby, C#, C, C++ dillerine ise orta ileri seviyede hakimim.
